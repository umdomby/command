Для рук/манипулятора (2 актуатора):
Один для "плеча" (наклон вверх/вниз для достижения чайника/ручки), второй для "локтя/хваталки" (сжатие пальцев или толчок). 
Тросики от актуатора тянут рычаг или сервоприводы на конце руки, чтобы "взять" кружку. VL53L5CX прикрепи на "ладонь" для автофокуса на объекте (измерь расстояние 5-30 см).
Пример: Сделай 2-суставную руку из алюминиевых профилей (типа 2020 экструзии). Трос от мотора обмотай вокруг шкива на суставе — вращение мотора тянет трос, сгибая руку. 
Для хватания добавь пассивный захват (из резины/пружины), который сжимается тросом.
Бонус: Для открытия двери — "рука" с резиновым наконечником толкает ручку вниз (линейное движение троса).

Для наклона тела/платформы (1-2 актуатора):
Один актуатор для передне-заднего наклона (баланс при "достижении" чайника, чтобы не опрокинуться). Тросы крепятся к "поясу" робота и тянут вниз/вверх, меняя центр тяжести.
Второй — для бокового наклона (стабилизация при повороте). Это поможет роботу "наклоняться" к свету/двери без колес. Крепи актуаторы под платформой, 
тросы через шкивы к каркасу.

Для ног (если добавишь, 1-2 актуатора):
Поскольку база на колесах, ноги можно сделать опциональными "амортизаторами" для неровностей (подъем/опускание по 10-20 см). Тросы от мотора тянут "колено" вверх для шага 
или стабилизации. Но если фокус на колесах, пропусти — сэкономь на руках.

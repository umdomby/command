Вывод: для триала 2 часа + периодических проверок JWT — это стандарт де-факто в 2025–2026 годах для десктопных приложений с онлайн-активацией.
Клиент (C# WinForms/WPF)  
↓ HTTPS (POST /api/trial, /api/validate)
Backend (Go 1.24 + Gin / Echo / chi)  
↔ PostgreSQL (таблица used_hwids + optional blacklist)
Admin + регистрация покупателей (Next.js 15 App Router)  
→ /api/... (Next.js API routes или тот же Go бэкенд)

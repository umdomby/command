Huawei P30 Pro (2019) — почти брат Mate 20 Pro: Leica quad-камера (5x zoom), отличный depth для портретов, но фронтальная без 3D (2D Face Unlock).
Samsung Galaxy S10+ / S20+ (старые, но похожи по экрану/камерам) — curved AMOLED, multiple камеры, но без 3D front.

Huawei Mate 20 Pro (Android 9–10 с EMUI) ты можешь передавать видео с камеры (включая портретный эффект с depth/bokeh) на ПК в реальном времени несколькими способами. 
Самые актуальные и работающие в 2026 году варианты — беспроводное зеркалирование экрана (Wireless Projection / Miracast) и scrcpy (по USB или Wi-Fi). Вот пошаговые инструкции.
1. Самый простой беспроводной способ: Wireless Projection (Miracast) на Windows 10/11
   Mate 20 Pro поддерживает Miracast — это встроенная функция, работает без доп. приложений.
   На ПК (Windows 10/11):

Убедись, что ПК подключён к Wi-Fi (та же сеть, что и телефон).
Нажми Win + K (или ищи "Connect" / "Подключить" в поиске Windows).
Если "Connect" нет: Параметры → Система → Проецирование на этот компьютер → Добавить функцию "Wireless Display" (если не установлено — скачай из Microsoft Store или Optional features).
Включи "Доступно везде" или "Доступно везде в защищённых сетях".

На Mate 20 Pro:

Свайп вниз → Настройки быстрого доступа → Wireless Projection (Беспроводная проекция / Easy Projection / Mirror).
Телефон найдёт твой ПК → выбери его и подключись.
Открой камеру (в портретном режиме для bokeh-эффекта) → видео с телефона будет в окне на ПК.

Задержка: 100–300 мс (приемлемо для распознавания объектов близко).
Качество: до 1080p, но иногда 720p (зависит от Wi-Fi).
2. Лучший вариант по качеству и задержке: scrcpy (бесплатно, open-source)
   Это даёт низкую задержку (~30–100 мс), управление с ПК и работает отлично на старых Huawei.
   Подготовка на телефоне:

Включи "Отладка по USB" (Настройки → О телефоне → тапай 7 раз по Build number → Параметры разработчика → USB debugging).
Разреши установку от неизвестных источников (если нужно).

На ПК:

Скачай scrcpy с официального GitHub: https://github.com/Genymobile/scrcpy (последняя версия v2.x или v3.x — работает в 2026).
Распакуй архив.
Подключи телефон по USB → разреши отладку на экране телефона.

Запуск:

Открой командную строку в папке scrcpy.
Команда: scrcpy (по USB).
Для Wi-Fi: сначала по USB выполни adb tcpip 5555, отключи кабель → scrcpy -e или adb connect IP_телефона:5555 (IP найди в настройках Wi-Fi).

Открой камеру на телефоне → на ПК увидишь зеркало экрана с live-видео (портретный эффект работает).
Захватывай окно в OBS/OpenCV для распознавания.

# Что такое Recovery (рекавери) — простыми словами (и точными) словами, 2025 год

| Параметр                  | Объяснение простыми словами                                                                                           | Для чего нужен на практике                                                                                     |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| **Recovery**              | Это **отдельная мини-система** на телефоне, которая живёт в отдельном разделе (`recovery`), независимо от Android.   | Когда основной Android «умер» (bootloop, кирпич, «The system has been destroyed»), рекавери может остаться живым и спасти телефон. |
| **Стоковое recovery**     | То, что ставит производитель (Xiaomi, Samsung, Huawei и т.д.). Очень простое.                                        | • Сброс до заводских (Factory Reset)  <br>• Установка официальных обновлений через sideload  <br>• Wipe cache |
| **Кастомное recovery**    | TWRP, OrangeFox, SHRP, PitchBlack и др.                                                                              | • Полный бэкап и восстановление всех разделов  <br>• Установка любых zip (прошивки, Magisk, GApps)  <br>• Mount системы, ADB в рекавери  <br>• Форматирование и ремонт разделов |
| Как выглядит на экране    | Стоковое — синий/чёрный экран с меню на английском/китайском <br>Кастомное — красивое сенсорное меню                  | —                                                                                                               |

### Сравнительная таблица 2025 года

| Возможность                          | Стоковое recovery | TWRP / OrangeFox |
|--------------------------------------|-------------------|------------------|
| Factory Reset / Wipe data            | Да                | Да               |
| Установка OTA через sideload         | Да                | Да               |
| Установка кастомных прошивок/GApps   | Нет               | Да               |
| Полный бэкап NANDroid (всей системы) | Нет               | Да               |
| Mount /system, /data, /vendor        | Нет               | Да               |
| ADB в рекавери (push/pull)           | Иногда            | Да               |
| Форматирование разделов, Fix NTFS    | Нет               | Да               |
| Root через Magisk прямо в рекавери   | Нет               | Да               |

### Как зайти в рекавери (самые частые комбинации)

| Бренд            | Комбинация кнопок (телефон выключен)      | Через ADB                     |
|------------------|-------------------------------------------|-------------------------------|
| Xiaomi / Redmi   | Vol+ + Power                              | `adb reboot recovery`         |
| Samsung          | Vol+ + Home + Power (старые) или Vol+ + Power + Bixby (новые) | `adb reboot recovery` |
| Realme / OPPO    | Vol+ + Power                              | `adb reboot recovery`         |
| Huawei / Honor   | Vol+ + Power (иногда нужно держать кабель) | `adb reboot recovery`         |
| Pixel / OnePlus  | Vol+ + Power                              | `adb reboot recovery`         |

### Коротко запомнить
- Recovery — это **«запасной парашют»** телефона.
- Если основной Android сломался — рекавери может его починить (если само рекавери живое).
- При надписи «The system has been destroyed» рекавери почти всегда тоже убито → остаётся только SP Flash Tool / MTK Client.

Если коротко:  
**Recovery = отдельная маленькая операционная система на телефоне для экстренного ремонта.**

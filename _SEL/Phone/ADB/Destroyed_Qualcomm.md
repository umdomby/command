# Qualcomm в EDL-режим (9008) — как перевести и чем прошить (2025)

EDL (Emergency Download Mode) = порт **Qualcomm HS-USB QDLoader 9008 (COMx)**  
Это «спасательный» режим Qualcomm. Даже полностью мёртвый кирпич (ничего не включается, не заряжается, чёрный экран) определяется как 9008 и прошивается за 3–10 минут.

| Способ перевода в EDL 9008                          | Когда работает                                                                 | Как делать (2025)                                                                                           | Шанс успеха |
|-----------------------------------------------------|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|-------------|
| 1. Команда из Fastboot (самый простой)              | Телефон хотя бы заходит в Fastboot                                             | `fastboot oem edl`   или   `fastboot reboot edl`                                                            | 95 %        |
| 2. Кнопками (без ПК)                                | На большинстве Xiaomi/Redmi/Poco на Snapdragon                                 | Выключить → зажать **Vol+ + Vol–** → вставить кабель → держать до появления 9008 в диспетчере              | 90 %        |
| 3. Кнопками + Power                                 | Samsung, OnePlus, Realme                                                       | Выключить → зажать **Vol– + Power** → вставить кабель (или Vol+ + Vol– + Power)                            | 85 %        |
| 4. Тест-поинт (100 % работает даже на полном кирпиче)| Телефон вообще не реагирует ни на что                                          | Открыть крышку → замкнуть две точки на плате иголкой/пинцетом → вставить кабель → отпустить                | 100 %       |
| 5. Программный способ из ADB                        | Телефон включён или в рекавери                                                 | `adb reboot edl`                                                                                            | 90 %        |
| 6. Через донгл (Hydra, UFI, Miracle и т.д.)         | Сервисный способ                                                               | В программе нажать кнопку «EDL» или «Reboot to EDL»                                                         | 100 %       |

### Что делать после появления 9008 в диспетчере устройств

| Программа                | Что прошивать                                | Где скачать (живая 2025)                                      |
|--------------------------|----------------------------------------------|----------------------------------------------------------------|
| QFIL (Qualcomm Flash Image Loader) | Официальные .mbn + .xml прошивки             | Входит в QPST → https://qpsttool.com                           |
| MiFlash (официальный от Xiaomi) | Fastboot-ROM для Snapdragon Xiaomi           | https://miflash.ru                                             |
| Hydra Tool / Pandora     | Автоматический unbrick (нажал кнопку — готово)| Платные донглы                                                 |
| Prog_EMMC_Firehose файлы + QFIL | Для самых сложных случаев (rawprogram0.xml + patch0.xml) | Ищутся по модели + «firehose»                                  |

### Самые частые модели 2024–2025, где это нужно
- Xiaomi: Redmi Note 8/9/10/11/12 Pro, Poco X3/X4/X5/X6, Redmi 10/12 и т.д.
- Realme, OPPO, OnePlus на Snapdragon 6xx–8xx
- Samsung A-серия (A50, A51, A52 и др.)

### Быстрый чек-лист (если телефон вообще не включается)
1. Зажать **Vol+ + Vol–**
2. Вставить кабель в ПК
3. В диспетчере устройств → Порты → появился **Qualcomm HS-USB QDLoader 9008** → готово!
4. Запустить QFIL/MiFlash → выбрать прошивку → Start

Если не появился — ищем тест-поинт по своей модели (Google: «модель + edl test point» — миллион фоток).

Готово — телефон оживёт за 5–10 минут.

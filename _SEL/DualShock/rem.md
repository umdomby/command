Да, на https://dualshock-tools.github.io/ (DualShock Calibration GUI) можно откалибровать триггеры L2/R2 DualShock 4. Это веб-инструмент (работает в Chrome/Edge), который использует заводские USB-команды Sony для перезаписи калибровки в NVS (память контроллера). Поддерживает стики + триггеры (опция "Calibrate L2/R2", beta, но работает).
Как откалибровать на dualshock-tools.github.io (пошагово):

Подключи DS4 по USB (Bluetooth не работает!).
Открой сайт в Chrome/Edge → Connect (выбери контроллер).
Выбери Calibrate L2/R2 (или полную калибровку).
Следуй инструкциям: отпусти → полностью нажми L2/R2 несколько раз (3–5).
Save changes permanently (сохранит в память навсегда).
Проверь: Gamepad Tester (gamepad-tester.com) — значения должны быть 0–255 линейно.

Предупреждения: Только оригинальный DS4 (детектит подделки). Резервная калибровка не сохраняется — рискуешь (редко brick). Тестировано на V1/V2.

В DS4Windows — ПОЧТИ калибровка триггеров (настройка, не полная ре-калибровка). Не перезаписывает аппаратную калибровку (как сайт), а эмулирует через профили: deadzone, sensitivity, anti-deadzone. Фиксит дрифт/неточности в 80% случаев (PC-игры).
Как настроить триггеры в DS4Windows (Ryochan7 fork, github.com/Ryochan7/DS4Windows):

Скачай/установи DS4Windows → подключи DS4 (USB/BT).
Edit Profile (твой контроллер) → Controller Readings (вкладка справа).
Настрой L2/R2:ПараметрДля дрифта/неточностейЗначениеDeadzoneИгнор слабого нажатия0.05–0.10Anti-Deadzone"Противо-дрифт"0.10–0.20SensitivityУсиление110–150%MaxПолный диапазон100%
Calibrate (в Other/Steering) — для гиро/стиков, но мониторь триггеры в реал-тайм.
Save → тест в игре/PC.

Скачай (если ещё не скачал)
https://sourceforge.net/projects/monitorswitcher/files/latest/download
Размер 80 Кб, распакуй в любую папку, запусти MonitorSwitcherGUI.exe
Создай профиль «ВСЁ ВКЛЮЧЕНО»


Включи ВСЕ 6 мониторов (Win+P → Extend, если надо).
Расставь их как обычно в «Параметры → Система → Дисплей».
Щёлкни правой кнопкой по иконке в трее → Save Profile → New Profile
Назови FULL → OK
(файл .xml появится в %AppData%\MonitorSwitcher\Profiles)


Создай профиль «ТОЛЬКО ИГРОВОЙ»


Перейди в «Параметры → Система → Дисплей»
Выдели по очереди 3 лишних монитора → «Отключить этот дисплей»
Оставь только основной (на 4070 SUPER, 144/165 Гц).
Снова правой кнопкой по иконке → Save Profile → New Profile
Назови GAME → OK


Назначь горячие клавиши (самое удобное!)


Правой кнопкой по иконке → Set Hotkeys
Выдели профиль FULL → в поле Hotkey жми Ctrl + Alt + F → OK
Выдели профиль GAME → жми Ctrl + Alt + G → OK

Готово!
Теперь:

Перед запуском Grid Legends → жмёшь Ctrl + Alt + G → 3 монитора гаснут за 1 сек, FPS +15–30 %.
Вышел из игры → Ctrl + Alt + F → всё возвращается ровно как было: разрешение, частота, позиция иконок — ничего не сбивается.

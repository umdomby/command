В таком пустом проекте настроить USB Virtual COM Port с нуля «руками» будет очень сложно, так как в нем нет необходимых драйверов от STMicroelectronics.

Чтобы у вас всё получилось легко и через графический конфигуратор, сделайте проект заново строго по этим шагам:

# Шаг 1: 
Правильное создание проекта с конфигуратором (.ioc)
В меню нажмите: File -> New -> STM32 Project.
В появившемся окне выбора контроллера (Target Selection) перейдите на вкладку MCU/MPU Selector.
В поле поиска Part Number введите: STM32F103C8
Выберите этот чип в списке снизу и нажмите Next.
Важно: В следующем окне укажите имя проекта (например, USB_CDC_Blink) и убедитесь, что в поле Targeted Project Type выбрано STM32Cube (а не Empty, как у вас получилось сейчас).
Нажмите Finish. Перед вами откроется графическое окно микросхемы (файл с расширением .ioc).

# Шаг 2: 
Настройка USB и частоты в графическом редакторе
Когда откроется графическая карта чипа, настройте три вещи:
Включаем внешний кварц (без него USB не заведется):
Слева выберите System Core -> RCC.
В строке High Speed Clock (HSE) выберите Crystal/Ceramic Resonator.
Включаем аппаратный USB:
Слева выберите Connectivity -> USB.
Поставьте галочку Device (FS).
Активируем виртуальный COM-порт:
Слева выберите Middleware and Software -> USB_DEVICE.
В выпадающем списке Class For FS IP выберите Communication Device Class (Virtual Port Com).
Настраиваем светодиод на плате:
Найдите на картинке чипа ножку PC13, кликните по ней и выберите GPIO_Output.
Настройка частоты (Clock Configuration):
Перейдите на вкладку Clock Configuration (сверху).
В поле Input Frequency (слева) должно стоять 8 MHz.
В блоке PLL Source Mux выберите точку HSE.
В поле HCLK (MHz) введите 72 и нажмите Enter. Среда сама пересчитает множители. Главное, убедитесь, что в поле To USB Peripheral (MHz) получилось ровно 48.
Нажмите Ctrl + S для сохранения. Среда спросит: "Do you want to generate Code?" — нажмите Yes.

# Шаг 3: 
Добавляем код в сгенерированный main.c
После генерации кода ваш файл main.c станет большим, в нем появится много готовых функций инициализации от ST.
Подключаем инклуды: В самом верху файла найдите секцию /* USER CODE BEGIN Includes */ и добавьте туда библиотеки:

```
/* USER CODE BEGIN Includes */
#include "usbd_cdc_if.h"  // Драйвер виртуального COM-порта
#include <stdio.h>        // Для сборки строки через sprintf
/* USER CODE END Includes */
Добавляем отправку в цикл: Прокрутите файл вниз до функции int main(void) и найдите бесконечный цикл while (1). Измените его так:

C
/* USER CODE BEGIN WHILE */
uint32_t counter = 0;   // Переменная счетчика
char buffer[32];        // Буфер для текста

while (1)
{
counter++; // Увеличиваем счетчик на 1

      // Формируем строку (например: "Counter: 5\r\n")
      int len = sprintf(buffer, "Counter: %lu\r\n", counter);

      // Отправляем строку в USB COM-порт
      CDC_Transmit_FS((uint8_t*)buffer, len);

      // Мигаем светодиодом PC13 для проверки работы
      HAL_GPIO_TogglePin(GPIOC, GPIO_PIN_13);

      // Задержка ровно 1 секунда (1000 мс)
      HAL_Delay(1000);

    /* USER CODE END WHILE */

    /* USER CODE BEGIN 3 */
}
/* USER CODE END 3 */
```

Внимание: Пишите код только внутри комментариев USER CODE BEGIN и USER CODE END, иначе при повторном открытии настроек куб сотрет ваши изменения.

# Шаг 4: 
Сборка, прошивка и проверка
Нажмите на иконку Молотка (Build) на панели, чтобы скомпилировать проект без ошибок.
Подключите ваш программатор ST-LINK V2 к плате и прошейте контроллер через кнопку Run (зеленая кнопка Play) или через утилиту ST-LINK Utility.
Важно: После того как плата прошилась, отключите программатор и подключите плату к компьютеру по обычному USB-кабелю через Micro-USB разъем на самой плате STM32.
Теперь откройте Диспетчер устройств Windows. В разделе Порты (COM и LPT) у вас появится новое устройство: STMicroelectronics Virtual COM Port (COM...).
Вы можете открыть любую программу-терминал (Putty, Serial Port Monitor или вкладку Terminal внутри STM32CubeIDE), подключиться к этому COM-порту на любой скорости, и вы увидите, как каждую секунду плата присылает увеличивающийся счетчик.

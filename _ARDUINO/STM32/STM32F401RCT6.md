STM32F401RCT6 (STM32F401CCU6)

# Шаг 1. 
Включаем кварцевый резонатор (HSE)Без внешнего кварца USB-загрузчик не поймет компьютер.
В левой колонке нажмите на вкладку System Core.В раскрывшемся списке выберите RCC.В появившемся меню посередине найдите строку 
High Speed Clock (HSE) и выберите вместо Disable пункт Crystal/Ceramic Resonator. На схеме чипа две ножки (PH0 и PH1) окрасятся в зеленый цвет.
# Шаг 2. 
Настраиваем светодиод для миганияНайдите на схеме чипа ножку, к которой у вас на плате подключен светодиод (на китайских платах Black 
Pill это обычно PC13):Найдите ножку PC13 на картинке микроконтроллера (она сверху слева, вторая от угла).Нажмите на неё левой кнопкой мыши.
В выпадающем списке выберите GPIO_Output. Ножка станет зеленой.
# Шаг 3. 
Переходим к генерации проектаВ самом верху окна перейдите на вкладку 
Project Manager (она находится правее вкладки Clock Configuration).Напишите имя проекта в поле Project Name (английскими буквами, например, 
F401_TypeC).В поле Toolchain / IDE выберите из выпадающего списка STM32CubeIDE.В правом верхнем углу нажмите синюю кнопку GENERATE CODE.
Когда генерация завершится, программа предложит открыть проект — нажмите Open Project, и он сам откроется в вашей STM32CubeIDE 
со всеми нужными файлами и конфигурацией.

на этапе настройки в STM32CubeMX включали HSE в режим Crystal/Ceramic Resonator, то перед генерацией кода нужно было зайти на вкладку 
Clock Configuration (сверху) и переключить источник тактирования на HSE / PLLCLK, чтобы частота ядра стала 84 МГц, а на шину USB пошло ровно 48 МГц.

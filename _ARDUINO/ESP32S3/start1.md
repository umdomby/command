ESP32-S3 имеет композитное USB-устройство с несколькими интерфейсами:
Interface 0 → обычно CDC (виртуальный COM-порт, COM11 у тебя) — для логов и иногда прошивки
Interface 2 (или MI_02) → именно JTAG-часть (это то, что нужно для esp-builtin)


Interface 2 → JTAG/debug — для отладки и специальной прошивки через OpenOCD (esp-builtin)

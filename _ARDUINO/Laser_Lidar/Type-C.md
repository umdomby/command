LiDAR и датчики типа VL53L5CX для смартфона: подключение по USB-C/Bluetooth и 3D-карты
Да, такие устройства существуют, но с нюансами: полноценные внешние LiDAR для смартфонов (Android/iOS) — редкость, чаще это портативные сканеры с USB-C (быстрее Bluetooth по скорости передачи данных). Bluetooth удобен для беспроводного, но медленнее (до 2–3 Мбит/с vs USB-C ~480 Мбит/с). Вывод 3D-карты "сразу на экран сверху" (в реальном времени, сверху вниз) возможен в apps вроде Polycam или PIX4Dcatch — они показывают point cloud или mesh в AR/3D-view. Для VL53L5CX (8×8 зон, ToF-датчик) прямых готовых модулей для смартфона нет, но можно адаптировать через USB-I²C адаптер; AI-модели для 3D-реконструкции — в основном на базе TensorFlow Lite для Android.

Операционная система — MetaOS (Dual-Domain)
Это не Windows и не TwinCAT!

Sinsegye разработала собственную промышленную двухдоменную ОС MetaOS (на базе Linux).
Архитектура: два независимых ядра/домена, работающих параллельно:
Real-time domain (жёсткий реал-тайм) — dedicated real-time kernel. Отвечает за PLC, Motion, EtherCAT, interrupts, deterministic tasks.
Non-real-time domain (soft real-time / general purpose) — обычный Linux kernel. Для HMI, Vision, AI, Web, CAM, desktop-приложений и т.д.

Поддерживаются стандартные IEC 61131-3 языки + SoftMotion / PLCopen Motion.

Инструмент управления: HyperCare (веб-интерфейс или приложение) — мониторинг, бэкап, восстановление, распределение ресурсов по доменам.
CODESYS + собственный runtime.

1 Dot Projector (лазерный проектор)
→ выпускает ~30 000 невидимых инфракрасных точек по специальному трафарету (DOE — дифракционный элемент).
2 Что считывает точки?
Инфракрасная камера (IR Camera / NIR Camera) — это отдельный сенсор в notch’е.
Она видит только инфракрасный свет, обычные цвета для неё не важны.
3 Что происходит дальше?
ИК-камера снимает картинку, на которой видно, как точки искажаются на твоём лице.
Процессор (в телефоне) сравнивает эту искажённую картинку с эталонным паттерном.
По степени искажения каждой точки считает расстояние (триангуляция) → получает карту глубины.


# structured light (структурированный свет) на базе ИК-проектора
- Dot projector (проектор точек) — специальный ИК-лазер (VCSEL) 
проецирует сетку из ~30 000 невидимых инфракрасных точек на лицо (или объект). Точки расположены в известном (калиброванном) порядке.
- ИК-камера (IR camera) — фиксирует, как эти точки искажаются на поверхности лица (деформация зависит от расстояния до каждой точки).
- Flood illuminator — дополнительный ИК-прожектор для равномерного освещения в темноте.
- TOF proximity sensor — помогает определять расстояние и активировать систему.
- RGB-камера — обычная фронтальная камера для цветного изображения (на Mate 20 Pro — 24 МП).

- По триангуляции (геометрия) для каждой точки рассчитывается глубина (depth) → создаётся карта глубины (depth map) и 3D-модель лица (point cloud + mesh).
- На iPhone это делает Neural Engine в Secure Enclave. На Mate 20 Pro — Kirin 980 + NPU (нейронный процессор).
- Результат: математическая модель лица (не просто фото), очень устойчивая к обману (маски, фото и т.д.).

Вы увидите 3D-модель (mesh + точки), которую можно вращать. Многие пользователи сканировали лица/головы и получали point cloud.

# Перенос данных по WiFi в C#
Через 3D Live Maker — приложение позволяет сканировать и экспортировать модель (OBJ, иногда point cloud). 
Потом по WiFi (через облако Huawei или просто файл) перекинуть в C# (используйте библиотеки типа Helix Toolkit, OpenTK, Unity и т.д. для отображения point cloud).

- Huawei предоставляет Camera2 API и AR Engine (AREngine) на новых устройствах. 
На Mate 20 Pro можно попробовать получить depth map через ARCore-подобные API или сторонние библиотеки.
- Полный raw-доступ к ИК-датчику ограничен (без root или специальных прошивок). 
- Но depth-данные доступны в AR-приложениях.C# (например, через MAUI/Xamarin или .NET для Android) можно писать приложение, 
которое захватывает depth frame и передаёт по WiFi (Socket/TCP или WebSocket) на ПК. На ПК — отображать как point cloud 
(библиотеки Point Cloud Library wrapper, Helix 3D и т.п.).

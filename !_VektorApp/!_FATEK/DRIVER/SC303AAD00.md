FA008 = PGCL1 - PGCL12 - PGCL13 - PGCL14 - PGCL15  -зафиксировалось только done на 5 - это нормально?
Коды PGCL (от Pulse Generator Clear) означают этапы очистки различных типов данных в памяти энкодера:
PGCL1 – PGCL13: Очистка логов ошибок, данных о перегрузках и внутренних счетчиков.
PGCL14 – PGCL15: Очистка данных о многооборотных значениях (Multi-turn data) — это самый важный этап для устранения ошибки .810.
Нажми F несколько раз → зайди в Utility Function (FAxxx)

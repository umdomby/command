# Реал-тайм 3D-реконструкция (основная часть)
Используй нейросетевые SLAM 2025–2026 годов. Они на feed-forward нейросетях (уже предобучены на миллиардах изображений):
- MASt3R-SLAM (CVPR 2025) — лучший выбор для тебя.
Работает в реальном времени (~15 FPS на RTX 40xx/50xx), строит плотную 3D-модель на лету из монокулярного видео. Использует сильные 3D-priors из MASt3R.
Поддерживает две камеры (multi-view).
- SLAM3R (CVPR 2025 Highlight) — ~20+ FPS, отличное качество dense reconstruction.
Прямо регрессирует 3D-точки из RGB без явной калибровки камеры.

Подключаешь обе камеры (GigE). Синхронизируешь кадры по timestamp (software sync). Подаёшь пары (left + right) в MASt3R-SLAM или SLAM3R.
В реальном времени получаешь:
Live 3D-модель (point cloud / mesh).
Трекинг позы. Возможность сразу считать кривизну поверхности для поиска вмятин.

Вывод:
Для двух обычных камер ИИ-модели типа MASt3R-SLAM и SLAM3R
